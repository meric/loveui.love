class (parent)
Creates a class using shallow copy. An instance of the class can be created by calling the class as a function. The class' __super key refers to its parent class, if any. The init function can be overridden to initialize instances of the class; Arguments provided when the class is called is passed to this init function.

Parameters

  • parent: [optional] The parent class to subclass from.

Return value:

The newly created class.
